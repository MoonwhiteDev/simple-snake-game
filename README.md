# Simple Snake Game

A classic Snake game implemented as a single HTML file using JavaScript and the HTML5 canvas.

## Game Rules

- Control the snake using the keyboard.
- Eat the food to grow longer.
- Avoid hitting the walls or the snakeâ€™s own body.
- The game ends when a collision occurs.

## How to Play

1. Open `index.html` in a modern web browser.
2. Use button "R" to start/stop the game.
3. Use the arrow keys to control the snake.
4. Try to achieve the highest score possible.

## Features

- Single-file HTML implementation
- No external libraries
- Smooth movement and collision detection
- Score tracking

## Implementation

- HTML5 Canvas for rendering
- JavaScript game loop and input handling
